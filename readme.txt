===============================
Sistema de Stock - Va Mates
===============================

Descripción:
------------
Este proyecto es un sistema de gestión de stock para el emprendimiento "Va Mates", desarrollado en PHP, con base de datos MySQL. 
Permite gestionar productos (mates, yerbas, bombillas, etc.), controlar ventas, compras, gastos y obtener métricas financieras clave en un panel principal.

Tecnologías utilizadas:
------------------------
- PHP (Backend)
- MySQL (Base de datos)
- HTML / CSS / JavaScript (Frontend)
- Bootstrap (Diseño responsivo)
- Chart.js (Gráficos del dashboard)

Funciones principales:
-----------------------
- Login de administrador
- Gestión de productos (ABM)
- Registro de compras y ventas
- Control de stock actualizado en tiempo real
- Visualización de métricas clave (ventas, ganancias, gastos)
- Reportes de productos más vendidos
- Control de estado (pendiente, pagada, cancelada / activo o desactivo)
- Calculador temporal de paquetes de yerba

Requisitos:
------------
- Servidor con Apache y PHP 7.4+
- MySQL o MariaDB
- Navegador actualizado

Instalación:
-------------
1. Clonar o descargar este repositorio.
2. Importar el archivo `mates_ventas.sql` en tu servidor MySQL.
3. Configurar la conexión a la base de datos en `config/config.php`.
4. Levantar el proyecto en tu servidor local (ej: XAMPP, Laragon).
5. Acceder al login desde `http://localhost/vamates3/login/login.php`.
